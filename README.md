# Calculator
A simple React-Redux Application

## Running Locally

Make sure you have [Yarn](https://yarnpkg.com/en/) and [NodeJS](https://nodejs.org/en/) installed.

```bash
git clone git@github.com:laraanna/Calculator.git
cd Calculator
yarn install
yarn start
```
